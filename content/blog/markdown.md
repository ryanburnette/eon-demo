## Escape Test

> It's easier to ask forgiveness \
> than it is to get permission \
> \- Grace Hopper

| Markdown                             | Render                       |
| ------------------------------------ | ---------------------------- |
| \*\*bold\*\*, \_italic\_, \~strike\~ | **bold**, _italic_, ~strike~ |
| em\*\*\*bold\*\*\*en                 | em**_bold_**en               |
| \`code\`                             | `code`                       |
| \`\` \`code\` \`\`                   | `` `code` ``                 |
| \\\\ \\\\ \\\\\\\\                   | \\ \\ \\\\                   |

```go
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

````md
```go
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
````
